Population and Internet Use

Have you ever wondered how many people for a given country use the internet? I catch myself thinking about this from time to time:

  • Does internet usage trend with population growth?
  • At what point will internet usage catch up to the population for a given country?
  • Which countries have have low/high populations of internet users?

Why You Might Be Interested

  • If you are a digital marketer, you might want to know which countries to enter.
  • Make inferences regarding technological infrastructure for a given country.
  • Better understand your own website metrics.
  • And more...

And The Good News Is...

Now you can see how these factors change over time for any country!

Just go to my app and enter the country code to see the results.
